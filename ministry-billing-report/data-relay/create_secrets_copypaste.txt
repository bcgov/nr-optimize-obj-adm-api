Full steps are here: https://github.com/franTarkenton/demo-oc-serviceaccount

Enter into CMD:
cd send-usage-emails
SET OC_URL_SRC=https://downloads-openshift-console.apps.silver.devops.gov.bc.ca/amd64/windows/oc.zip
curl %OC_URL_SRC% -o oc.zip
tar -zxvf oc.zip
rm oc.zip
SET PATH=%cd%;%PATH%

Go to OCP, get personal oc login command with token, sign into OCP on command line.

Create service accounts with:
oc process -f serviceAccountTemplate.yaml --param-file=service_account_params.txt | oc create -f -

Go to OCP, under administrator > user management > service account, get the token for the new account

SET SA_KEY=<key goes here>
SET CLUSTER_URL=https://api.silver.devops.gov.bc.ca:6443
oc login --token=%SA_KEY% --server=%CLUSTER_URL%
oc port-forward postgresql-6-lm694 5432:5432

Run in background with:
start "oc-port-forward" cmd /k "SET PATH=%cd%;%PATH% & oc port-forward postgresql-6-lm694 5432:5432"

Kill background task with: 
taskkill /FI "WindowTitle eq oc*" /T /F

Login to metabase with:
"C:\Program Files\PostgreSQL\10\bin\psql.exe" -h localhost -p 5432 -d metabase -U postgres
NOTE: psql.exe should work if it's on my PATH
